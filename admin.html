🚀 How to Update the Main Site

Step 1: Add your new tracks using the form above
Step 2: Click "Copy JSON for index.html"
Step 3: Open index.html in a text editor (Notepad on Windows)
Step 4: Press Ctrl+F and search for const TRACKS = [
Step 5: Select from the [ all the way to the matching ]; and paste your new JSON
Step 6: Save the file and upload to HostGator — done!
